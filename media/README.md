## ⚡ 快速开始

### 添加照片

将照片放入 `media/photos/` 目录，建议：
- 格式：JPG/PNG
- 分辨率：1920x1080 以下
- 命名：`01_description.jpg`

### 添加视频

**方案A：Git LFS（推荐）**
```bash
git lfs install
git lfs track "*.mp4"
git add media/videos/*.mp4
git commit -m "Add demo videos"
```

**方案B：外部链接**
将视频上传到B站/YouTube，在上方表格中添加链接。

---

## ⚠️ 注意事项

- 照片建议使用 WebP 格式减小体积
- 视频建议压缩后上传，或使用外部链接
- 遵循项目许可证，不要上传侵犯版权的内容
